SISTEMA WEB ADMINISTRATIVO PARA LA GESTION DE CASOS EN 
LA EMPRESA FIGUEROA & ASOCIADOS.


Este es un sistema sencillo hecho con la plantilla AdminLTEv3.0.5, y con 
tecnologias tales como php,jquery,mysql,ajax,json,y boostrap 4.

Funciones:

-registro de usuarios.
-registro de personal de la empresa.
-registro de asistencias.
-Seguimiento de casos.
-almacenamiento y gestion de documentos.
-area de recursos compartidos.
